# 🛍️ Galerie Luxora

Une galerie de produits moderne, responsive et interactive développée avec HTML5, CSS3 et JavaScript Vanilla.

## 📖 Présentation

Galerie Luxora est une application web statique permettant de présenter une collection de produits sous forme de cartes interactives.

Ce projet met en pratique les fondamentaux du développement Front-End :

- Structure HTML5 sémantique
- Mise en page responsive avec CSS3
- Manipulation du DOM avec JavaScript
- Expérience utilisateur moderne
- Déploiement via GitHub Pages

# ⚙️ Installation

## Cloner le dépôt

```bash

```

## Accéder au projet

```bash
cd galerieluxora
```

## Ouvrir le projet

Ouvrir simplement le fichier :

```
index.html
```

dans votre navigateur.
