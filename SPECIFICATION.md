# 📋 Spécifications fonctionnelles

# Galerie Luxora

---

# 1. Présentation du projet

Galerie Luxora est une application web permettant de présenter une collection de produits dans une interface moderne, responsive et intuitive.

Le projet a été réalisé dans le cadre d'un exercice de développement Front-End afin de consolider les compétences fondamentales en HTML, CSS et JavaScript.

---

# 2. Contexte

Ce projet s'inscrit dans un parcours d'apprentissage du développement web.

Il vise à mettre en pratique :

- la structuration HTML ;
- la mise en page CSS ;
- les interactions JavaScript ;
- les bonnes pratiques de développement.

---

# 3. Objectifs

## Objectif principal

Créer une galerie de produits moderne, responsive et facilement maintenable.

## Objectifs secondaires

- pratiquer le DOM
- pratiquer le Responsive Design
- améliorer l'organisation du code
- préparer un projet de portfolio

---

# 4. Public cible

Le site s'adresse :

- aux visiteurs souhaitant découvrir une collection de produits ;
- aux recruteurs souhaitant évaluer les compétences Front-End ;
- aux développeurs débutants recherchant un exemple pédagogique.

---

# 5. Fonctionnalités

## Affichage des produits

Chaque produit est présenté sous forme de carte contenant :

- image
- nom
- description
- prix
- bouton d'action (si présent)

---

## Responsive Design

Le site s'adapte automatiquement aux :

# 15. Conclusion

Galerie Luxora constitue un projet d'entraînement permettant de démontrer les compétences fondamentales en développement Front-End.

Il met en évidence la capacité à concevoir une interface responsive, organiser un projet web, utiliser Git pour le versionnement et déployer une application statique avec GitHub Pages.

Ce projet constitue une base solide pouvant évoluer vers une véritable application e-commerce avec une API, un système de recherche, un panier et une gestion des utilisateurs.
